# sails-api-server
 Generate quick pure RESTful API with sailsjs 1.x.x + MySQL + passport and jwt token authentication


## NOTE AFTER CLONE
 - change jwttokensecret in `config/tokenSecret` (.gitignore it for protection)
 - Change MySQL URL in `dev` and `production` env to reflect your own environment
 - Change DataEncryptionKeys for models in `config/models`
